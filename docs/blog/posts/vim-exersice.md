## 基础步骤

| 按键                                                | 作用         |
|-----------------------------------------------------|--------------|
| ++"h"++                                             | 向左移动光标 |
| ++"j"++                                             | 向下移动     |
| ++"k"++                                             | 向上移动     |
| ++"l"++                                             | 向右移动     |
| ++"i"++                                             | 进入插入模式 |
| ++escape++  /  ++ctrl+c++  /  ++ctrl+bracket-left++ | 返回正常模式 |

```练习区域
          k
          |
    h <-  O  -> l
          |
          j
```

## 快速按单词移动

| 按键           | 作用                     |
|----------------|--------------------------|
| ++"w"++        | 移动到下一个单词的开头   |
| ++"b"++        | 移动到上一个单词的开头   |
| ++"e"++        | 移动到下一个单词的结尾   |

| ++"g"++ ++e++  | 移动到上一个大单词的结尾 |
